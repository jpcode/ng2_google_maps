## Usage

### Basic
Simple google maps feature:
 - Geolocation with a pool request of 50 per seconds
 - Clustering Strategy
 - Infowindows

### Quick start
**Make sure you have Node version >= 5.0 and NPM >= 3**

```bash
# change directory to our repo
cd dir_name

# install the repo with npm
npm install

# start the server
npm start
